    Notch filter to eliminate noise from an ECG signal

    Designed a 110 Hz notch filter using pole-zero placement in MATLAB to remove power-line interference from ECG signals. Conducted FFT analysis to evaluate signal clarity, demonstrating practical DSP filter design for biomedical applications.

    Microcontroller-based Energy Efficient Escalator using Arduino Uno

    Built an Arduino-controlled escalator prototype that activates only when passengers are detected. Integrated sensors and relays to control motor operation, ensuring energy efficiency and user safety through microcontroller-based automation.

    Optimum Power Control at Different Loading Using Current Injector Device

    Developed a power balancing system using current injectors, relays, and MCBs for smart substation control. This setup optimized load distribution across phases, enhancing power system reliability and operational efficiency.


    |Google Drive Link


    PID Controller Design for Electric Traction Motor

    Modeled and tuned PID controllers in MATLAB/Simulink to improve electric traction motor performance. Applied pole-zero analysis, root locus methods, and transfer function modeling for stable and responsive control system behavior.


    |Google Drive Link


    Different CMOS SRAM Circuits' Implementation and Performance Analysis -Cadence Virtuoso

    Designed and simulated 7T–10T CMOS SRAM circuits in Cadence Virtuoso. Evaluated performance metrics such as delay, power consumption, and area, gaining practical experience in VLSI design, layout, and optimization.

    SG3525 PWM Inverter 12V to 220V, 300W using two IRF-520 transistors

    Constructed a PWM inverter circuit using the SG3525 IC and IRF-520 transistors to convert 12V DC to 220V AC. Focused on PWM control, efficient transistor switching, soldering techniques, and performance testing.

    Created a 2D flat layout in AutoCAD, incorporating structural details and complete electrical load design. Included load calculation, cable sizing, MCB/RCB selection, and standard-compliant switchboard placement for residential planning.
